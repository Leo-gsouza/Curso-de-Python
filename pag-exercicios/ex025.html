Exercicio 25

Crie um programa que leia o nome de uma pessoa e diga se ela tem “SILVA” no nome.

                    
            

nome = str(input('Nome completo: ')).title().strip()

print('Silva' in nome)

if 'Silva' in nome:
    print(f'{nome} tem Silva')
else:
    print(f'{nome} não tem Silva')

Anotações

Para resolver esse exercicio foi usado:

  1. Strip » Para remover os espaços antes e depois da informação digitada pelo usuario
  2. Title » Vai deixar a string digitada com apenas a primeira letra maiuscula, para não interferir na comparação, pois letras maiusculas e minusculas fazem diferença
  3. IN » o comando IN tem como objetivo procurar algo em. No exercicio ele procurou 'Silva' dentro da variavel nome, caso encontre retorna True, caso contrario False
