Exercicio 19

Faça um programa que leia um ângulo qualquer e mostre na tela o valor do seno, cosseno e tangente desse ângulo.

                


    from random import choice

    aluno1 = str(input('Nome do 1º aluno: '))
    aluno2 = str(input('Nome do 2º aluno: '))
    aluno3 = str(input('Nome do 3º aluno: '))
    aluno4 = str(input('Nome do 4º aluno: '))

    alunos = [aluno1 , aluno2, aluno3, aluno4 ]
    escolhido = choice(alunos)

    print(f'O aluno escolhido foi {escolhido}')

Anotações

Para sortear um valor dentro de uma lista dá para usar o metodo choice importado da biblioteca random.
from random import choice.

Para criar uma lista usa-se o conchetes [].
