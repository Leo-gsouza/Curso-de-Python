Exercicio 22

Crie um programa que leia o nome completo de uma pessoa e mostre:

  1. O nome com todas as letras maiúsculas e minúsculas.
  2. Quantas letras ao todo (sem considerar espaços).
  3. Quantas letras tem o primeiro nome.

                    
            

nome = str(input('Nome completo: ')).capitalize().strip()

print(f'Nome em letra maiuscula » {nome.upper()}')
print(f'Nome em letra minuscula » {nome.lower()}')
print(f'Seu nome tem {len(nome) - nome.count(" ")} letras')
print(f'Seu primeiro nome tem {nome.find(" ")} letras ')

""" USANDO LISTA PARA SEPARAR O NOME """

nome_separado = nome.split()

print(f'Seu primeiro nome é {nome_separado[0]} 
       e tem {len(nome_separado[0])} letras')

Anotações

Foram usados seis comandos neste exercicio

UPPER() » Deixa todas as strings em maiusculo. Para utiliza-lo basta inserir ele após a variavel, exemplo: variavel.upper()

LOWER() » Deixa todas as strings em minusculo. Para utiliza-lo basta inserir ele após a variavel, exemplo: variavel.lower()

LEN() » Dá a quantidade de caracteres contida na variavel. Para utiliza-lo basta inserir ele antes e colocar a variavel entre parenteses, exemplo: len(variavel)

COUNT() » Contar o número de vezes que determinado caractere aparece em uma variavel. Para utiliza-lo basta inserir ele após a variavel, exemplo: variavel.count('inserir o caractere que deseja contar')

FIND() » Encontrar a primeira posição de um determinado caractere dentro da variavel. Para utiliza-lo basta inserir ele após a variavel, exemplo: variavel.find('inserir o caractere que deseja encontrar')

Lista para separar o nome

SPLIT() » O comando split cria uma lista. Cada caractere separado por espaço é transformado em um item dentro de uma lista. Vamos supor que a variavel nome contenha o valor = Ana Luisa Silva. Será criado uma lista [Ana, Luisa, Silva] con três itens. Para utiliza-lo basta inserir após a variavel, exemplo: variavel.split()

observação: É recomendado criar outra variavel, exemplo: outra_variavel = (variavel.split). Desta forma essa outra variavel vai ter uma lista e a variavel original será preservada
