Exercicio 27

Faça um programa que leia o nome completo de uma pessoa, mostrando em seguida o primeiro e o último nome separadamente.

                    
            

nome = str(input('Nome completo: ')).strip().title().split()

primeiro_nome = nome[0]
ultimo_nome = nome[-1]

print(f'Seu primeiro nome é {primeiro_nome}\
    \nSeu ultimo nome é {ultimo_nome}')

Anotações

Para resolver esse exercicio foi usado:

  1. split » O comando split transforma a string em uma lista usando os espaços como referencia. Exemplo: Entre João Almeida existe um espaço separando, o comando variavel.split() vai criar uma lista = [João, almeida] com dois itens dentro.
  2. variavel[0] » [0] representa o primeiro item dentro da lista
  3. variavel[-1] » [-1] representa o ultimo item dentro da lista
