Exercicio 20

O mesmo professor do desafio 19 quer sortear a ordem de apresentação de trabalhos dos alunos. Faça um programa que leia o nome dos quatro alunos e mostre a ordem sorteada.

                


    import random

    aluno1 = str(input('Nome do 1º Aluno: ')).capitalize()
    aluno2 = str(input('Nome do 2º Aluno: ')).capitalize()
    aluno3 = str(input('Nome do 3º Aluno: ')).capitalize()
    aluno4 = str(input('Nome do 4º Aluno: ')).capitalize()
    
    alunos = [aluno1, aluno2, aluno3, aluno4]
    ordem = random.shuffle(alunos)
    
    print(f'A ordem sorteada pelo professor foi {alunos}')

Anotações

Para sortear a ordem dentro da lista foi usado o metodo shuffle importado também da biblioteca random.
import random.

Observação: Note que neste exercicio foi importado so a biblioteca random e dentro do código foi usado random.shuffle(lista). Mas poderia ter importado o metodo direto from random import shuffle, facilitando o uso do metodo dentro do codigo shuffle(lista)
