Exercicio 24

Crie um programa que leia o nome de uma cidade diga se ela começa ou não com o nome “SANTO”.

                    
            

cidade = str(input('Nome da cidade: ')).strip().title()

print(cidade[0:5] == 'Santo')

""" Condição SE """

if cidade[:5] == 'Santo':
    print(f'Sim! A cidade {cidade} começa com Santo')
else:
    print(f'Não! A cidade {cidade} não começa com Santo')

Anotações

Para resolver esse exercicio foi usado:

  1. Strip » Para remover os espaços antes e depois da informação digitada pelo usuario
  2. Title » Vai deixar a string digitada com apenas a primeira letra maiuscula, para não interferir na comparação, pois letras maiusculas e minusculas fazem diferença
  3. cidade[:5] » Para pegar os 5 primeiros caracteres digitados
  4. cidade[:5] == 'Santo' » Comparar se os cinco primeiros digitos é igual a Santo
  5. No primeiro caso vai retonar true ou False apenas, já no segundo que foi colocado a condição SE, vai retonar a mensagem digitada na condição
