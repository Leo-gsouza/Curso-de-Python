Exercicio 26

Faça um programa que leia uma frase pelo teclado e mostre quantas vezes aparece a letra “A”, em que posição ela aparece a primeira vez e em que posição ela aparece a última vez.

                    
            

frase = str(input('Digite uma frase: ')).strip().lower()
frase = frase.replace(' ', '')


letra_a = 'a' in frase

if letra_a:
print(f'A frase contém {frase.count("a")} letras A\
  \nA 1ª letra A se encontra na {frase.find("a")+1}ª posição \
  \nA ultima letra A se encontra na {frase.rfind("a")+1}ª posição')
else:
print(f'A frase digitada não contem a letra "a" .')

Anotações

Para resolver esse exercicio foi usado:

  1. lower » Para deixar todas as letras minusculas e não ter problema com a digitação do usuario
  2. Replace » O comando replace substitui algo por outro, neste caso substituimos o espaço por nada, pois os espaços sáo contabilizados pelo count e isto iria gerar problema no resultado
  3. variavel letra_a » foi criada para ter um valor boleano true ou false para dar continuidade no exercicio usando a condição SE, se existir "a" na frase seguimos por um lado, se não existir vamos por outro.
  4. count » o comando count contabiliza o numero de caracteres informado entre o parenteses variavel.count('busca').
  5. find » O comando find vai encontra a primeira posição do caracetre informado variavel.find('busca')
  6. rfind » O comando rfind vai encontra a ultima posição do caracetre informado variavel.rfind('busca')
  7. Note que depois do find e rfind aparece um +1 que tem como objetivo somar + um com resultado encontrado pelo comando, devido ao fato da contagem iniciar do 0. exemplo: A letra R na palavra amar está na 4º posição para nós, mas para linguagem ela está na terceira, pois se inicia do zero » A=0, M=1, A=2, R=3. somando + 1 o resultado apresentado será 3 + 1 = 4, que para nós é a posição exata
