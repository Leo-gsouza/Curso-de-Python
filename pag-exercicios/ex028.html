Exercicio 28

Escreva um programa que faça o computador “pensar” em um número inteiro entre 0 e 5 e peça para o usuário tentar descobrir qual foi o número escolhido pelo computador. O programa deverá escrever na tela se o usuário venceu ou perdeu.

                    
                        

from random import randint

computador = randint(1,5)

jogador = int(input('Digite um número de 1 a 5 » '))

if computador == jogador:
    print(f'Venceu!\
            \nComputador » {computador}\
            \nJogador » {jogador}')
else:
        print(f'Perdeu!\
            \nComputador » {computador}\
            \nJogador » {jogador}')

Anotações

Para resolver esse exercicio foi usado:

  1. randint » O comando randint da biblioteca random faz um sorteio entre os numeros inseridos entre parenteses, exemplo: randint(5, 100). Será sorteado um valor entre 5 e 100 e aplicado a variavel
  2. Condição SE » Se o numero sorteado for igual( == )ao digitado pelo jogador terá como resultado venceu, caso contrario perdeu
